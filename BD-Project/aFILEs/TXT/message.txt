**Entidades:**

* Personagem (idPersonagem)
* Item (idItem)
* Arma (idArma)
* Armadura (idArmadura)
* Talismã (idTalismã)
* Feitiço/Magia (idFeitiçoMagia)
* Inimigo (idInimigo)
* Local (idLocal)
* Chefe (idChefe)

**Relacionamentos:**

* **Personagem:**
    * Usa (Usa) -> Item
    * Equipa (Equipa) -> Arma
    * Veste (Veste) -> Armadura
    * Usa (Usa) -> Talismã
    * Lança (Lança) -> Feitiço/Magia
    * Visita (Visita) -> Local
    * Combate (Combate) -> Inimigo
    * Derrota (Derrota) -> Chefe
* **Item:**
    * É (É) -> Arma
    * É (É) -> Armadura
    * É (É) -> Talismã
    * É (É) -> Feitiço/Magia
* **Inimigo:**
    * Habita (Habita) -> Local
* **Chefe:**
    * Protege (Protege) -> Local
    * Dropa (Dropa) -> Item

**Cardinalidades:**

* Personagem:
    * Usa (Usa): N:M com Item
    * Equipa (Equipa): 1:N com Arma
    * Veste (Veste): 1:N com Armadura
    * Usa (Usa): N:M com Talismã
    * Lança (Lança): N:M com Feitiço/Magia
    * Visita (Visita): N:M com Local
    * Combate (Combate): N:M com Inimigo
    * Derrota (Derrota): N:M com Chefe
* Item:
    * É (É): 1:1 com Arma
    * É (É): 1:1 com Armadura
    * É (É): 1:1 com Talismã
    * É (É): 1:1 com Feitiço/Magia
* Inimigo:
    * Habita (Habita): N:M com Local
* Chefe:
    * Protege (Protege): 1:N com Local
    * Dropa (Dropa): 1:N com Item

**Atributos:**

* Personagem:
    * idPersonagem (chave primária)
    * Nome
    * Classe
    * Nível
    * Atributos (Vigor, Força, Destreza, etc.)
* Item:
    * idItem (chave primária)
    * Nome
    * Descrição
    * Tipo
    * Efeito
    * Valor
* Arma:
    * idArma (chave primária)
    * Tipo (Espada, Machado, Arco, etc.)
    * Dano
    * Alcance
    * Atributos de Escalonamento
    * Habilidades
* Armadura:
    * idArmadura (chave primária)
    * Tipo (Armadura leve, pesada, etc.)
    * Defesa física
    * Defesa mágica
    * Resistência a elementos
    * Peso
* Talismã:
    * idTalismã (chave primária)
    * Nome
    * Descrição
    * Efeito
    * Requisitos de uso
* Feitiço/Magia:
    * idFeitiçoMagia (chave primária)
    * Nome
    * Descrição
    * Tipo de dano
    * Custo de mana
    * Requisitos de atributos
* Inimigo:
    * idInimigo (chave primária)
    * Nome
    * Descrição
    * Aparência
    * Localização
    * Atributos
    * Ataques
    * Fraquezas
    * Drops
* Local:
    * idLocal (chave primária)
    * Nome
    * Descrição
    * Área
    * Pontos de interesse
    * Inimigos (relacionado a idInimigo)
    * Itens (relacionado a idItem)
* Chefe:
    * idChefe (chave primária)
    * Nome
    * Descrição
    * Aparência
    * Localização
    * Atributos
    * Ataques
    * Fraquezas
    * Drops (relacionado a idItem)
